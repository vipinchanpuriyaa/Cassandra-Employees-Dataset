2.	Employee_Train_Dataset.csv A curated 45,000-row subset used for machine learning training.

**Regression Tasks:**

1.Predict Required Salary Hike to Improve Employee Satisfaction
2.Predict Which Employees Should Be Promoted (Role-based Forecasting)
